# SignalForge Agent Guide

SignalForge is the working codename for the Design Partner Insight Engine, a Gitwit-aligned portfolio artifact focused on startup/product discovery and design-partner insight synthesis.

Use repo-local Codex skills from `.agents/skills`.

## Product Boundary

Keep work centered on React + TypeScript, NestJS + TypeScript, MongoDB/Mongoose, and an AI provider abstraction for design-partner insight workflows.

Do not drift into industrial maintenance, equipment troubleshooting, manuals, CMMS, technician copilots, diagnostics, field service, machinery, PlantBrain, or Tenzin overlap.

## Current Scaffold

This repository currently contains only initial Codex/agent scaffolding. Do not add application code, dependencies, package files, Docker files, database setup, or framework scaffolding until explicitly requested.
